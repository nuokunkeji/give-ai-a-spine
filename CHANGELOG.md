# Changelog

All notable changes to Spine are documented here.

## [0.1.0] - 2026-07-10

### Added

- `reality-check` skill with light, standard, and audit response depths.
- Native Codex and Claude Code marketplace manifests.
- Dependency-free installer for Cursor, Gemini CLI, OpenCode, Codex, and Claude Code.
- Conflict protection, dry runs, backups, and idempotent installation.
- Twenty-four public evaluation cases and twelve paired Codex CLI runs.
- Cross-platform CI, schema checks, local-link validation, and secret scanning.
- English and Chinese documentation, demonstration GIF, launch copy, and contribution templates.
